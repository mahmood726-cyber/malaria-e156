# Additional Files Plan

## Recommended Upload Set

- Additional file 1: `africa_benchmark.csv`
  - Format: `.csv`
  - Title: Africa-site malaria benchmark records
  - Description: Trial-level extracted variables for the `80` Africa-site malaria studies used in the benchmark analysis.

- Additional file 2: `africa_transfer_shortlist.csv`
  - Format: `.csv`
  - Title: Africa-site malaria transfer shortlist
  - Description: Subset of benchmark studies meeting the predefined shortlist criteria for lean operational footprint and prospective registration.

- Additional file 3: `malaria_deep_dive.csv`
  - Format: `.csv`
  - Title: Malaria deep-dive outcome and leadership features
  - Description: Study-level variables derived from the results section and registry-based leadership heuristic.

- Additional file 4: `summary.md` and `malaria_deep_dive_summary.md`
  - Format: convert to `.pdf` or `.docx` before upload if preferred
  - Title: Supplemental benchmark summaries
  - Description: Human-readable summaries of the main benchmark and malaria-specific deep dive.

- Additional file 5: `code/`
  - Format: `.zip`
  - Title: Reproducibility code bundle
  - Description: Scripts used to query ClinicalTrials.gov, score study records, and generate the malaria deep-dive outputs.

## Manuscript Citations

- Cite the public GitHub repository in the data-availability statement.
- If the authors mint a Zenodo DOI, replace the GitHub-only citation with the archived DOI.
- If the authors archive the exact search on searchRxiv, cite that DOI in Methods and the data-availability statement.
